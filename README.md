# ice-warehouse-deeds
A centralized collection of all deed documents for the warehouses purchased by ICE in 2026. They are provided for archival purposes.

# Provenance
These documents were compiled by Project Salt Box through county tax assessor portals and/or state public records requests, and public reporting.
